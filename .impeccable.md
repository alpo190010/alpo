# Design Context — PageScore

## Target Audience
Shopify store owners, e-commerce sellers, DTC brand operators. Busy, practical people who care about revenue. Not designers, not developers — merchants who want to sell more product. Age 25-45, mobile-first, scan not read.

## Use Cases
1. Paste a product page URL → get a conversion score in 30 seconds
2. See exactly where the page is losing sales (visual diagnostic)
3. Get actionable fixes they can implement today
4. Optionally get email report for detailed fixes

## Brand Personality / Tone
- **Confident** — like a trusted advisor, not a salesperson
- **Sharp** — cuts through noise, says exactly what matters
- **Premium but accessible** — looks like it costs money, works like it's free
- **Analytical** — data-driven, specific, never vague
- **2026 modern** — not 2020 SaaS template, not 2024 dark mode

## Aesthetic Direction
- Clean white/warm backgrounds, NOT clinical
- One accent color: electric blue (#2563EB)
- Inter + JetBrains Mono for score numbers
- Generous whitespace, tight content
- Cards with subtle borders, no heavy shadows
- Feels like: Stripe meets Linear meets a Bloomberg terminal for Shopify

## What Makes It Memorable
The score reveal moment — number counts up, revenue impact hits, user feels "I need to fix this." The diagnostic loader that tells you exactly what it's checking. The leak cards that are so specific the user thinks "how does it know my page?"

## Quality Bar
Flagship — this is the entire product. Every pixel matters.

## Technical Constraints
- Next.js 14+ App Router
- Tailwind CSS available but inline styles also used
- Must be responsive (390px mobile to 1440px desktop)
- Performance: no heavy JS libraries, CSS animations preferred
- Accessibility: WCAG AA minimum
